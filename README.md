<h3>short description of my project</h3>
<p>My project is to make a tamagotchi following the Fallout tutorial. Fallout is a YSWS by Hack Club, hosting a 7 day hackathon in Shenzen. There is an OLED screen, with 3 buttons controlling what the Tamagotchi does and a buzzer to output sounds.</p>

<h3>why i made the project </h3> 
<p>This is my first hardware project, and I really hope that I will be able to make lots of cool hardware, learn new skills and qualify for the hackathon in Shenzen.</p>
